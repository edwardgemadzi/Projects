Task 2: Installing SASS:Provide step-by-step instructions for installing Sass on your computer. You can choose your preferred method (e.g., CLI, GUI tools).

  1. Install node.js and npm
  2. Install SASS using npm: type 'npm install -g sass' in the command prompt
  3. Run 'sass --version' to verify the installation
  4. Compile a .scss file into a .css file: type 'sass filename.scss filename.css'
  5. Enable watch mode: type 'sass --watch filename.scss:filename.css'

Task 4: Exploring Operators in Sass: Explain how these operators can be used to perform calculations within Sass

Sass provides mathematical operators that allow calculations directly within stylesheets. These include:

  • Addition (+): Used to add values (e.g., width: 100px + 20px;).
  • Subtraction (-): Used to subtract values (e.g., margin: 50px - 10px;).
  • Multiplication (*): Used to scale values (e.g., font-size: 16px * 1.5;).
  • Division (/): Used to divide values (e.g., line-height: 20px / 2;). Sass requires careful usage to avoid conflicts with CSS division.
  • Modulo (%): Used to get the remainder of a division (e.g., $result: 10 % 3; results in 1).
